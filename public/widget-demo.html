Zapta Chat Widget Demo

This page demonstrates how to embed the Zapta chat widget on any website.

Important:

Replace YOUR_AGENT_ID_HERE with your actual agent ID from the Zapta dashboard.

Installation Code

Add this code snippet before the closing </body> tag:

<script>
window.ZAPTA_AGENT_ID = 'YOUR_AGENT_ID_HERE';
window.ZAPTA_API_URL = 'https://zapta-nu.vercel.app';
window.ZAPTA_PRIMARY_COLOR = '#FF7A59'; // Optional
</script>
<script src="https://zapta-nu.vercel.app/zapta-widget.js"></script>
